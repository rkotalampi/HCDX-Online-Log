Column
 How to use the column
Category
 Please select the log category from the pulldown menu
Frequency
 Frequency as kHz. For the FM station you can use MHz, too
+
 Mark the checkbox if this is the first time the station have been heard anywhere, by anyone (DXer).
Tent.
 Please mark this if the station is tentative. This means that you are certain about the identity of the station but because you never heard an ID you can't be 100%.
Station
 Please add the station name, QTH, call etc necessary to identify the station.
Month, Day, Year
 Please select from pulldown menu when did you hear the station (in UTC). If you heard the station during the midnight UTC please choose the first day.
*
 These checkboxes are to indicate that the station started or ended it's transmission. The log will have the time undelined.
Start (UTC)
 Please type the time in UTC when you started to listen to the station. If you only listened it for a shortwhile or unsure about when you stopped to listen, please only use Start (UTC) entry and leave End (UTC) empty.
End (UTC)
 Please type the time in UTC when you stopped to listen to the station.
Comments
 If you have any comments about the program, station, conditions etc please type it here.
Listener
 How would you like the system to identify the listener of the logging.
